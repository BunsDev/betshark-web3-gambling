### Dev

Production

```bash
# -- install dependencies --
npm install

# -- build --
pnpm build

# -- run --
pnpm run start
```

Client

```bash
# -- develop --
pnpm dev
```

### Stack

-   Svelte (For pride month)
-   Express (For quick server serving)
-   Vite (Crazy fast build speed)
-   pnpm (Best package manager)
